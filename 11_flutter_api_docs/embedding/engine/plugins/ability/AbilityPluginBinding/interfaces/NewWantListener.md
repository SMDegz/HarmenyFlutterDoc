# Interface: NewWantListener

Delegate interface for handling new wants on behalf of the main UIAbility.

## Methods

### onNewWant()

> **onNewWant**(`want`, `launchParams`): `void`

Called when a new want is started for the UIAbility.

#### Parameters

##### want

`Want`

The new want that was started for the UIAbility

##### launchParams

`LaunchParam`

The launch parameters

#### Returns

`void`
