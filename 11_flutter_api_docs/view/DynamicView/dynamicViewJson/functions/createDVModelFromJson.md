# Function: createDVModelFromJson()

> **createDVModelFromJson**(`json`): [`DVModel`](../../dynamicView/classes/DVModel.md)

Creates a DVModel instance from a JSON object.
This function recursively parses the JSON structure to build a complete DVModel tree.

## Parameters

### json

`Object`

The JSON object representing the view structure

## Returns

[`DVModel`](../../dynamicView/classes/DVModel.md)

A DVModel instance created from the JSON
